1.1 Overview of Web Technologies

Web development means creating websites and web applications that run in a web browser. A website is made using different web technologies. The main technologies are:

  • Web Browser
  • HTML
  • CSS
  • JavaScript

These technologies work together to create a complete website.

What is a Web Browser?

A web browser is software used to open and view websites.

Examples of web browsers: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, Opera.

A browser reads HTML, CSS, and JavaScript and shows the webpage to the user.

Example: When you open www.google.com, the browser loads the webpage and displays it.

HTML (HyperText Markup Language)

HTML is used to create the structure of a webpage. It tells the browser:

  • What headings to show
  • Where to place images
  • How to create tables, forms, and paragraphs
HTML Structure
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Welcome</h1>
    <p>This is my first webpage.</p>
</body>
</html>
Rendered Result

CSS (Cascading Style Sheets)

CSS is used to design and style the webpage. It controls: Colors, Fonts, Borders, Layout, Background. Without CSS, webpages look plain.

CSS Example
<style>
body {
    background-color: lightblue;
}

h1 {
    color: red;
}
</style>
<h1>Styled Heading</h1>
Rendered Result

JavaScript

JavaScript is used to make webpages interactive. It can show alerts, change text, validate forms, and respond to button clicks.

JavaScript Interaction
<script>
function showMessage() {
    alert("Hello Student");
}
</script>

<button onclick="showMessage()">Click Me</button>
Rendered Result

How They Work Together

Technology Purpose
HTML Creates structure
CSS Adds design
JavaScript Adds interaction

Example: A login page uses HTML for boxes, CSS for style, and JS to check if fields are empty.

1.2 Client-Server Architecture

Web applications work using a client-server model.

What is a Client?

The client is the user's device or browser (Mobile phone, Laptop, Desktop). The client sends requests to the server.

What is a Server?

The server is a computer that stores website files and sends them to users. It processes requests and sends responses.

Example: When a user searches on Google, the browser sends a request to Google's server, which processes it and sends results back.

The Working Process

  1. User opens website in browser
  2. Browser sends request to server
  3. Server finds the required page or data
  4. Server sends response back
  5. Browser displays the webpage

Diagram

Client (Browser) ---- Request ----> Server
Client (Browser) <--- Response ---- Server

Advantages

  • Easy to manage
  • Data is stored in one place
  • Multiple users can use the application
  • Easy to update the website

1.3 Application Development

Application development means creating a web application. A professional web app consists of three interconnected layers. Think of it as the Showroom, the Office, and the Warehouse of a business.


Frontend
(The Interface)

Backend
(The Logic)

Database
(The Storage)

1.3.1 Frontend

The visible part of a website. It is what users see and interact with (Buttons, Menus, Forms).

Tools: HTML, CSS, JavaScript

1.3.2 Backend

The hidden part on the server. It handles logic like checking passwords and processing data.

Tools: PHP, Node.js, Python

1.3.3 Database

Used to store information securely (Names, Marks). Without it, data cannot be saved.

Tools: MySQL, MongoDB

Relationship Example: 1. User fills registration form (Frontend) → 2. Data is sent to server to check (Backend) → 3. User details are saved in a table (Database).

1.4 Basic Concept of Responsive Design

Responsive design means creating a website that works properly on all screen sizes.

The website should look good on:

  • Mobile phone
  • Tablet
  • Laptop
  • Desktop

Why Responsive Design is Needed

Today people use different devices. If a website is not responsive:

  • Text may become too small
  • Images may go outside the screen
  • Buttons may not work properly

Responsive design solves these problems.

Features of Responsive Design

  • Flexible layout
  • Resizable images
  • Mobile-friendly menus
  • Text adjusts automatically

Example of Responsive CSS

CSS Media Query
<style>
body {
    font-size: 20px;
}

@media screen and (max-width: 600px) {
    body {
        font-size: 34px;
    }
}
</style>
<body>
    <p>This text changes size on small screens!</p>
</body>
Meaning
• On large screens, text size is 20px
• On small screens, text size becomes 34px
<body> <h1>I change color on Mobile!</h1> </body>
Rendered Result

Example of Responsive Website

Flexbox Layout
<!DOCTYPE html>
<html>
<head>
<style>
.container {
    display: flex;
}

.box {
    width: 50%;
    background-color: lightblue;
    padding: 20px;
}

@media screen and (max-width: 600px) {
    .container {
        flex-direction: column;
    }

    .box {
        width: 100%;
    }
}
</style>
</head>
<body>

<div class="container">
    <div class="box">Box 1</div>
    <div class="box">Box 2</div>
</div>

</body>
</html>
