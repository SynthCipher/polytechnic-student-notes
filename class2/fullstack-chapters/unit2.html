2.1 HTML5

HTML5 is the latest version of HTML. It is used to create the structure of webpages.

2.1.1 Basic Structure of an HTML Document

HTML Structure
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome</h1>
    <p>This is my webpage.</p>
</body>
</html>
Explanation
  • <!DOCTYPE html> tells the browser that this is an HTML5 document.
  • <html> is the root tag.
  • <head> contains title and other information.
  • <body> contains visible content.

2.1.2 HTML Elements and Tags

An HTML element consists of: Opening tag, Content, Closing tag.

Example: <p>Hello World</p>

Here:
• <p> = opening tag
• Hello World = content
• </p> = closing tag

Some tags do not need closing tags. Example: <br>, <hr>, <img>

2.2 CSS Basics

CSS stands for Cascading Style Sheets. It is used to style webpages.

2.2.1 Introduction to CSS

CSS controls: Colors, Fonts, Spacing, and Layout.

2.2.2 Inline, Internal and External CSS

  • Inline CSS: Written inside the tag. <p style="color:red;">Hello</p>
  • Internal CSS: Written inside <style> tag in the head.
  • External CSS: Written in a separate .css file. <link rel="stylesheet" href="style.css">

2.2.3 CSS Selectors

Selectors
<style>
/* Element Selector */
p { color: red; }

/* Class Selector */
.highlight { color: blue; }

/* ID Selector */
#title { color: green; }
</style>

<p>Normal text</p>
<p class="highlight">Class styled</p>
<h1 id="title">ID styled</h1>
Rendered Result

2.2.5 Box Model

Every HTML element is like a box. The box model has 4 parts: Content, Padding, Border, Margin.

Box Model Code
<style>
.box {
    background-color: lightblue;
    padding: 20px;
    border: 5px solid red;
    margin: 30px;
}
</style>

<div class="box">Content goes here</div>
Rendered Result

2.2.7 CSS Display and Positioning

  • display: block - Block elements take full width (e.g., <div>, <p>).
  • display: inline - Inline elements take only needed width (e.g., <span>, <a>).
  • display: inline-block - Works like inline but width and height can be set.

2.3.6 Client Side Form Validation

Form Validation
<form onsubmit="return validate()">
    <input type="text" id="name" placeholder="Enter Name">
    <input type="submit">
</form>

<script>
function validate() {
    let name = document.getElementById("name").value;
    if(name == "") {
        alert("Name cannot be empty");
        return false;
    }
    alert("Success!");
    return false; // Prevents actual submission for demo
}
</script>

2.4 Bootstrap (Responsive Design)

Bootstrap is a CSS framework. It helps create responsive websites quickly by providing ready-made CSS classes, buttons, forms, and grid systems.

2.4.2 Grid Layout in Bootstrap

Bootstrap uses: container, row, and col.

Bootstrap Grid
<!-- Include Bootstrap CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/.../bootstrap.min.css">

<div class="container">
    <div class="row">
        <div class="col-6 bg-primary text-white p-3">Left</div>
        <div class="col-6 bg-success text-white p-3">Right</div>
    </div>
</div>
