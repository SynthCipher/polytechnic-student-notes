Method Overloading

Concept

Method Overloading means having many methods with the SAME name, but they take different types of data (like numbers or words).

Real-life Example: Think of the word "Cut". You can "Cut" paper with scissors, "Cut" hair with a trimmer, or "Cut" a cake with a knife. The action is called the same (Cut), but the tools are different.

Basic Code Example

// Filename: OverloadingDemo.java
class Calculator {
    // Method to add two integers
    int add(int a, int b) {
        return a + b;
    }
    
    // Overloaded method to add three integers
    int add(int a, int b, int c) {
        return a + b + c;
    }
}

public class OverloadingDemo {
    public static void main(String[] args) {
        Calculator cal = new Calculator();
        System.out.println("Addition of 2 numbers: " + cal.add(10, 20));
        System.out.println("Addition of 3 numbers: " + cal.add(10, 20, 30));
    }
}

Expected Output

Int Addition: 30
Double Addition: 35.5

1. Overload area() method to calculate area of Circle, Rectangle, and Triangle.
2. Overload volume() for Sphere and Cube.
3. Overload a method to display different types of data (int, string, char).
4. Overload a constructor for a student class.
