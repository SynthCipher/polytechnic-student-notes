PRACTICAL 02

Data Types & Control Structures

Concept

Data Types are like different sized containers to store different things (numbers, letters). Control Structures (if-else) help the computer make decisions.

Real-life Example: You use a bottle for water and a bag for books. You cannot put water in a bag! Similarly, an int stores numbers, not words.

Basic Code Example

// Filename: DataTypesDemo.java
public class DataTypesDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;
        int sum = a + b;
        
        System.out.println("Value of a: " + a);
        System.out.println("Value of b: " + b);
        
        if(sum > 25) {
            System.out.println("The Sum is greater than 25");
        } else {
            System.out.println("The Sum is less than 25");
        }
        
        System.out.println("Final Result: " + sum);
    }
}

Expected Output

Sum is greater than 25
Result: 30
