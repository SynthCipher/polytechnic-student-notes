PRACTICAL 06

Bank Account Management

Concept

Inheritance is a child getting properties from a parent. Static is something that is shared by everyone in the class, not just one person.

Real-life Example: In a bank, every person has a different "Account Balance", but the "Bank Name" (like SBI) is the same for everyone. So, Bank Name is static.

Basic Code Example

// Filename: BankDemo.java
class Account {
    static String bankName = "State Bank"; // Shared by all objects
    double balance;
    
    void deposit(double amount) {
        balance += amount;
        System.out.println("Amount Deposited: " + amount);
    }
}

class SavingsAccount extends Account {
    void showBalance() {
        System.out.println("Bank: " + bankName);
        System.out.println("Current Balance: " + balance);
    }
}

public class BankDemo {
    public static void main(String[] args) {
        SavingsAccount sa = new SavingsAccount();
        sa.deposit(5000.0);
        sa.showBalance();
    }
}

Expected Output

Deposited: 5000.0
Bank: State Bank
Current Balance: 5000.0
