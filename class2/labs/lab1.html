PRACTICAL 01

Environment Setup & Hello World

Concept

To run Java, we need a Translation Kit (JDK). It takes our English-like code and turns it into a secret language (Bytecode) that the computer can run.

Real-life Example: It is like a translator. If you speak Ladakhi and your friend speaks English, you need a translator to help you understand each other.

Basic Code Example

// Filename: HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        // This is a simple Java program
        System.out.println("Hello, Welcome to Java Programming!");
    }
}

Expected Output

Hello, Welcome to Java Programming!

Lab Assignment

Install JDK on your system, configure the environment variables (PATH), and write a program to display your Name, Roll Number, and Semester in separate lines.
