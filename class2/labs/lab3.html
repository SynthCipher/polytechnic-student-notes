PRACTICAL 03

Labelled Loops

Concept

A loop repeats a task. A Labelled Loop allows us to give a name to a loop so we can stop a "Big Task" completely from inside a "Small Task".

Real-life Example: Imagine searching for a specific toy in 5 different boxes. As soon as you find the toy in Box 2, you stop searching Box 2 AND you stop looking at the other 3 boxes too.

Basic Code Example

// Filename: LabelledLoop.java
public class LabelledLoop {
    public static void main(String[] args) {
        // Label for the outer loop
        outer: 
        for(int i=1; i<=3; i++) {
            for(int j=1; j<=3; j++) {
                if(i==2 && j==2) {
                    System.out.println("Breaking outer loop at i=2, j=2");
                    break outer; 
                }
                System.out.println("i=" + i + ", j=" + j);
            }
        }
    }
}

Expected Output

i=1, j=1
i=1, j=2
i=1, j=3
i=2, j=1

Lab Assignment

Develop 2 programs to explore labelled loops:
1. Use a labelled loop to find the first prime number in a range that also contains specific unwanted factors.
2. Search for a specific value in a 2D matrix and break the entire search process using a label once found.
