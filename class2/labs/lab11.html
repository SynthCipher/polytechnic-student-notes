PRACTICAL 11

Types of Inheritance

Concept

Inheritance is when a class gets functions and variables from another class. This helps us reuse code instead of writing it again.

Real-life Example: You inherit your surname, property, and house from your parents. You don't have to build a new house if you already have one from your parents.

Basic Code Example

// Filename: InheritanceDemo.java
class Parent {
    void show() { 
        System.out.println("This is the Parent Class method."); 
    }
}

class Child extends Parent {
    void display() { 
        System.out.println("This is the Child Class method."); 
    }
}

public class InheritanceDemo {
    public static void main(String[] args) {
        Child obj = new Child();
        obj.show();    // Accessing parent method
        obj.display(); // Accessing child method
    }
}

Expected Output

Parent Class
Child Class
