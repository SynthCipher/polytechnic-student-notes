PRACTICAL 12

Packages & Access Modifiers

Concept

Packages are like folders for classes. Visibility Modifiers (public, private) control who can see or use your code.

Real-life Example: Your diary is private (only you can see it). Your school textbook is public (anyone can read it). Your family photos are protected (only your family can see them).

Basic Code Example

// Filename: VisibilityDemo.java
class AccessTest {
    public int publicData = 10;
    private int privateData = 20;
    protected int protectedData = 30;

    void showPrivate() {
        System.out.println("Private Data (accessible inside class): " + privateData);
    }
}

public class VisibilityDemo {
    public static void main(String[] args) {
        AccessTest obj = new AccessTest();
        System.out.println("Public Data: " + obj.publicData);
        System.out.println("Protected Data: " + obj.protectedData);
        // System.out.println(obj.privateData); // Error: private access
        obj.showPrivate();
    }
}
