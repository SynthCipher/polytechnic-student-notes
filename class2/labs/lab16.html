PRACTICAL 16

Shape & Geometry Hierarchy

Concept

We can build complex families of shapes. A Circle is a 2D shape, and a Cylinder is like many circles stacked on top of each other.

Real-life Example: A "Square" is a flat shape. But if you take 6 squares and join them, you get a "Cube" (like a Ludo dice).

Basic Code Example

// Filename: ShapeHierarchyDemo.java
class Shape {
    String name;
    Shape(String n) { name = n; }
}

class Circle extends Shape {
    double radius;
    Circle(double r, String n) { 
        super(n); 
        this.radius = r; 
    }
    double area() { return Math.PI * radius * radius; }
}

public class ShapeHierarchyDemo {
    public static void main(String[] args) {
        Circle c = new Circle(5.0, "MyCircle");
        System.out.println("Shape Name: " + c.name);
        System.out.println("Circle Area: " + c.area());
    }
}

Expected Output

Shape: MyCircle
Area: 78.5398...

Lab Assignment

Create a class hierarchy: Shape (Base) -> Circle, Square. Then extend Circle to create Cylinder and Sphere, and Square to create Cube. Implement Area() and Volume() methods where applicable.
