PRACTICAL 08

String Methods

Concept

A String is just a line of text, like a name, a password, or a sentence. Java has many tools to change or check these strings.

Real-life Example: Your Name is a string. If you write your name "jigmat" and change it to "JIGMAT", you are using a string method (toUpperCase).

Basic Code Example

// Filename: StringMethodsDemo.java
public class StringMethodsDemo {
    public static void main(String[] args) {
        String str = "Java Programming";
        
        System.out.println("Original String: " + str);
        System.out.println("In Uppercase: " + str.toUpperCase());
        System.out.println("Length of String: " + str.length());
        System.out.println("Character at index 5: " + str.charAt(5));
        System.out.println("Contains 'Java': " + str.contains("Java"));
    }
}

Expected Output

Uppercase: JAVA PROGRAMMING
Length: 16
Contains 'Java': true

Lab Assignment

Design a program that performs the following string operations:
1. Check if two strings are equal (ignoring case).
2. Reverse a given string without using built-in reverse functions.
3. Change the case of a string (Toggle Case: e.g., 'Java' to 'jAVA').
