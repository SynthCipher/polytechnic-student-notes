PRACTICAL 04

Working with Arrays

Concept

An Array is like a collection of same things kept together. Instead of making 10 variables for 10 marks, we make 1 array called 'marks'.

Real-life Example: Like an egg tray. It is one object, but it can hold many eggs. Each hole has a position (index) starting from 0.

Basic Code Example

// Filename: ArrayDemo.java
public class ArrayDemo {
    public static void main(String[] args) {
        // Declaring and initializing an array
        int[] numbers = {10, 20, 30, 40, 50};
        
        System.out.println("Array elements are:");
        // Using for-each loop to print elements
        for(int x : numbers) {
            System.out.println("Element: " + x);
        }
    }
}

Expected Output

Array elements:
10 20 30 40 50

Lab Assignment
