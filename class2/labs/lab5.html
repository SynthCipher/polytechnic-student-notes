PRACTICAL 05

Using 'this' and 'super'

Concept

'this' means "This class" or "Me". 'super' means "Parent class" or "My Father". We use them to avoid confusion between similar names.

Real-life Example: You have your own shirt (this.shirt), and you might also wear your brother's jacket (super.jacket). Using these words makes it clear whose clothes you are talking about.

Basic Code Example

// Filename: RectangleBoxDemo.java
class Rectangle {
    double l, b;
    Rectangle(double l, double b) {
        this.l = l; // 'this' refers to current class variable
        this.b = b;
    }
    void displayArea() {
        System.out.println("Rectangle Area: " + (l * b));
    }
}

class Box extends Rectangle {
    double h;
    Box(double l, double b, double h) {
        super(l, b); // 'super' calls parent constructor
        this.h = h;
    }
    void displayVolume() {
        System.out.println("Box Volume: " + (l * b * h));
    }
}

public class RectangleBoxDemo {
    public static void main(String[] args) {
        Box myBox = new Box(10, 5, 5);
        myBox.displayArea();
        myBox.displayVolume();
    }
}

Expected Output

Area: 50.0
Volume: 250.0

Lab Assignment

Write a program to create a class Rectangle and a subclass Box. Calculate the area of the rectangle and the perimeter, then calculate the volume of the box. Use this to initialize current class members and super to initialize inherited members.
