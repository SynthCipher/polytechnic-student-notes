Multithreading in Java

Concept

Multithreading means doing many tasks at the same time. Instead of waiting for one task to finish, the computer works on many tasks together.

Real-life Example: Like a chef in a restaurant. He can boil water, chop vegetables, and talk to a waiter all at the same time. If he did only one thing at a time, the food would take forever!

Basic Code Example

// Filename: MultiThreadDemo.java
class MyThread extends Thread {
    public void run() {
        try {
            for(int i=1; i<=3; i++) {
                System.out.println(Thread.currentThread().getName() + " printing: " + i);
                Thread.sleep(500); // Wait for half a second
            }
        } catch (InterruptedException e) {
            System.out.println("Thread Interrupted");
        }
    }
}

public class MultiThreadDemo {
    public static void main(String[] args) {
        MyThread t1 = new MyThread();
        MyThread t2 = new MyThread();
        
        t1.setName("Thread-A");
        t2.setName("Thread-B");
        
        t1.start(); // Start first thread
        t2.start(); // Start second thread
    }
}

Expected Output

Thread-0: 1
Thread-1: 1
Thread-0: 2
Thread-1: 2...
