PRACTICAL 19

Nested Try, Throw & Throws

Concept

Throw is when we "shout" that an error happened. Throws is a warning that a function might cause an error. Nested Try is putting one try block inside another.

Real-life Example: If you find a fire in the building, you "throw" an alarm. The building's sign "Caution: Fire Hazard" is like a "throws" warning.

Basic Code Example

// Filename: NestedExceptionDemo.java
public class NestedExceptionDemo {
    // Method using throws
    static void checkAge(int age) throws ArithmeticException {
        if(age < 18) {
            // Explicitly throwing an exception
            throw new ArithmeticException("Age is less than 18, not eligible.");
        } else {
            System.out.println("Eligible for voting.");
        }
    }

    public static void main(String[] args) {
        try {
            // Outer try block
            try {
                // Inner try block
                checkAge(15);
            } catch(ArithmeticException e) {
                System.out.println("Caught in Inner Block: " + e.getMessage());
            }
        } catch(Exception e) {
            System.out.println("Caught in Outer Block: " + e);
        }
        System.out.println("Program finished.");
    }
}

Expected Output

Not eligible
