Responsive Design (Bootstrap)

Concept

Bootstrap is a kit of ready-made designs. It helps our website look good on both mobile phones and big computers automatically using a Grid System.

Real-life Example: Like using "Readymade Clothes". You don't have to sew the shirt yourself; you just pick a size (class) and wear it. It fits perfectly!
