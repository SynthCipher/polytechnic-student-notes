FULL STACK PRACTICAL 07

Node.js Server Application

Concept

Node.js is a platform to run JavaScript on a server. We can create a Server that listens for requests and sends responses (like HTML or Data).

Real-life Example: Like a phone operator. When you dial a number (Request), the operator connects you to the right person (Response).

Node.js Server Code

Filename: server.js
const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/html'});
  
  if(req.url === '/') {
      res.write('<h1>Welcome to Home</h1>');
  } else if(req.url === '/contact') {
      res.write('<h1>Contact: 999000111</h1>');
  } else {
      res.write('<h1>404 Not Found</h1>');
  }
  res.end();
});

server.listen(3000, () => {
  console.log('Server is running on port 3000');
});
Terminal Output
$ node server.js
Server started...

[Request] GET /contact
[Response] 200 OK
