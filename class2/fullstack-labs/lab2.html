Demonstration of CSS Techniques

Concept

Flexbox is a CSS tool used to arrange items in a row or a column easily. It helps us center things or spread them out evenly without using complex math.

Real-life Example: Like people sitting on a bench. If the bench is "Flex", you can tell people to sit in the middle, sit far apart, or sit all to one side very easily.

Basic Code & Live Preview

Filename: style.css
<style>
  /* Box Model & Colors */
  .box {
    width: 200px;
    padding: 20px;
    margin: 10px;
    border: 5px solid #6366f1;
    background-color: #e0e7ff;
    color: #1e293b;
    border-radius: 15px;
  }

  /* Flexbox Layout */
  .flex-container {
    display: flex;
    justify-content: center;
    gap: 20px;
  }

  /* Positioning */
  .fixed-top {
    position: sticky;
    top: 0;
    background: white;
  }
</style>
