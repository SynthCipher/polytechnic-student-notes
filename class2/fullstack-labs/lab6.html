PHP Basics & Operators

Concept

PHP is a server-side language. It runs on the server (like a hidden brain) and creates the HTML that the user sees. We use it for logic and math.

Real-life Example: Like a kitchen in a restaurant. You (the user) are in the dining hall. The PHP is the chef in the kitchen who takes your order and prepares the food (data).

Basic Code & Output

Filename: test.php
<?php
  $name = "Jigmat";   // String
  $age  = 21;         // Integer
  $isStudent = true;  // Boolean
  $marks = 85.5;      // Float

  // Arithmetic Operators
  $a = 10; $b = 3;
  echo "Sum: " . ($a + $b);
  echo "Product: " . ($a * $b);
  echo "Modulo: " . ($a % $b);

  // Logical Operator
  if($age > 18 && $isStudent) {
      echo "Adult Student";
  }
?>
Browser Output

Sum: 13

Product: 30

Modulo: 1

Adult Student
