PHP Arithmetic & Database

Concept

A webpage can take numbers from a user, do math (Add, Sub), show the result immediately on the same page, and then save the record in a database table.

Real-life Example: Like a calculator that also saves your history. You add 5 + 5 = 10, and the calculator remembers that calculation for later.

Arithmetic & Save Code

Filename: calc.php
<?php
if(isset($_POST['n1'])) {
  $n1 = $_POST['n1'];
  $n2 = $_POST['n2'];
  $op = $_POST['op'];
  $res = 0;

  if($op == '+') $res = $n1 + $n2;
  if($op == '*') $res = $n1 * $n2;

  echo "<h2>Result: $res</h2>";

  // Save to Database
  $conn = mysqli_connect("localhost", "root", "", "lab_db");
  $sql = "INSERT INTO history VALUES($n1, $n2, '$op', $res)";
  mysqli_query($conn, $sql);
}
?>
