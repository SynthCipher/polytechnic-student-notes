FULL STACK PRACTICAL 08

PHP Server Application

Concept

A PHP Server-side App uses PHP to handle URL requests and fetch data from a database. It allows different content to be shown on different URLs.

Real-life Example: Like a library. You ask for a book title (Request), the librarian finds it on the shelf (Database), and gives it to you (Response).

PHP Request Handling Code

Filename: index.php
<?php
// Database connection
$conn = mysqli_connect("localhost", "root", "", "my_db");

// Simple Routing
$page = $_GET['id'] ?? 1;

$sql = "SELECT * FROM posts WHERE id = $page";
$result = mysqli_query($conn, $sql);
$post = mysqli_fetch_assoc($result);

echo "<h1>" . $post['title'] . "</h1>";
echo "<p>" . $post['content'] . "</p>";
?>
Browser Output (?id=1)

About My School

Our school was founded in 1995 and has 500 students.

Lab Assignment
