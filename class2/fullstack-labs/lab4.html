Form Validation (JS)

Concept

Form Validation means checking if the user filled the form correctly (e.g., Is the email real? Is the password long enough?). We use JavaScript to do this before the data is sent.

Real-life Example: Like a security guard at a gate. He checks if you have an ID card and if you are wearing a mask before letting you in.

Basic Code & Live Preview

Filename: forms.html
<script>
function validate() {
  let name = document.getElementById("name").value;
  let email = document.getElementById("email").value;
  let age = document.getElementById("age").value;
  let phone = document.getElementById("phone").value;
  let pass = document.getElementById("pass").value;

  if(name == "" || email == "" || age == "" || 
     phone == "" || pass == "") {
      alert("All fields are required!");
      return false;
  }
  if(pass.length < 6) {
      alert("Password must be 6 characters!");
      return false;
  }
  alert("Validation Successful!");
  return true;
}
</script>
