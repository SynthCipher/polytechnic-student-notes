Comprehensive Full-Stack Project

Concept

The Comprehensive Project is where you use all your skills (HTML, CSS, JS, PHP, MySQL) to build a real-world application with a Frontend, a Backend, and Database storage.

Real-life Example: Like building a "College Management System" or an "E-Commerce Store". It has products, users, orders, and a database to remember everything.

Project Structure

A standard Full-Stack Project should have these files saved in your C:/xampp/htdocs/project/ folder:

  • /assets (Images, CSS, JS)
  • index.php (The landing page)
  • db.php (Database Connection)
  • login.php (User Login)
  • register.php (User Registration)
  • dashboard.php (The Main Content)

Lab Assignment

Final Project: Build a "Student Marks Management System".
1. Create a login page.
2. After login, show a table of students and their marks from MySQL.
3. Add a form to "Add New Student" marks into the table.
