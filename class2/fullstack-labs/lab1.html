Demonstration of All HTML Tags

Concept

HTML (HyperText Markup Language) is used to create the structure of a webpage. Think of it as the "Bones" of a building. Each tag has a special job, like showing a heading, a list, or a table.

Real-life Example: Like a school notebook. You have a Title (Heading), Paragraphs (Text), Bullet points (Lists), and Forms (Registration section).

Tag Guide (How they work)

Tag Work (Function)
<h1> - <h6>Used for Headings (h1 is biggest, h6 is smallest).
<p>Used for writing paragraphs of text.
<b>, <i>, <u>Formatting: Bold, Italic, and Underline.
<ul>, <ol>, <li>Unordered (bullets) and Ordered (1,2,3) lists.
<a>Creates a Link to another page.
<img>Displays an Image.
<table>, <tr>, <td>Used to show data in rows and columns.
<form>, <input>Used to take input from the user (like name, age).

Complete Code Demo

Filename: faculty_profile.html
<!DOCTYPE html>
<html>
<head>
  <title>Jigmat Dorjey | Faculty Profile</title>
</head>
<body>
  <!-- 1. Semantic Headers -->
  <header>
    <h1>Faculty Biography</h1>
    <h2>Jigmat Dorjey</h2>
    <p><mark>Lecturer in CSE Department</mark></p>
  </header>

  <!-- 2. Formatting Tags -->
  <p>
    <strong>Expertise:</strong> Full Stack Development <em>(MERN)</em><br>
    <small>Office Hours: 10:00 AM - 4:00 PM</small><br>
    Formula Example: H<sub>2</sub>O or E=mc<sup>2</sup>
  </p>

  <!-- 3. Multimedia -->
  <img src="../../assets/jigmat.jpg" alt="Profile" width="100">
  <br>
  <a href="https://jigmatdorjey.onela.in" target="_blank">Official Website</a>

  <!-- 4. Lists (Nested) -->
  <h3>Subjects Taught</h3>
  <ol>
    <li>Web Technology
      <ul>
        <li>HTML/CSS</li>
        <li>JavaScript</li>
      </ul>
    </li>
    <li>Java Programming</li>
  </ol>

  <!-- 5. Interactive Elements -->
  <details>
    <summary>Click to see Research Interests</summary>
    <p>Artificial Intelligence, Web Security, and Data Science.</p>
  </details>

  <!-- 6. Forms (Advanced) -->
  <h3>Student Query Form</h3>
  <form>
    <label>Student Name:</label> <input type="text"><br>
    <label>Gender:</label> 
    <input type="radio" name="g"> M <input type="radio" name="g"> F<br>
    <label>Department:</label>
    <select>
      <option>CSE</option>
      <option>Civil</option>
    </select><br>
    <label>DOB:</label> <input type="date"><br>
    <input type="submit" value="Submit Query">
  </form>

  <!-- 7. Quotes -->
  <blockquote>"Education is the most powerful weapon."</blockquote>
  <address>Govt. Polytechnic, Leh, Ladakh</address>
</body>
</html>
