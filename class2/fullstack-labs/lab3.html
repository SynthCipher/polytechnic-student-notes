FULL STACK PRACTICAL 03

Static Business Webpage

Concept

A Static Webpage is a website that looks the same for everyone. We use HTML for the "Bones" (structure) and CSS for the "Skin" (style/colors).

Real-life Example: Like a printed business card or a brochure. It gives information but it doesn't change when you look at it.

Basic Code & Live Preview

Filename: portfolio.html
<!DOCTYPE html>
<html>
<head>
  <style>
    body { font-family: Arial; margin: 0; }
    .navbar { background: #333; color: white; padding: 15px; }
    .hero { padding: 60px; text-align: center; background: #6366f1; color: white; }
    .content { padding: 30px; display: grid; grid-template-columns: 1fr 1fr; gap: 20px; }
    .card { border: 1px solid #ccc; padding: 15px; border-radius: 10px; }
    footer { background: #333; color: white; text-align: center; padding: 10px; }
  </style>
</head>
<body>
  <div class="navbar">My Portfolio</div>
  <div class="hero">
    <h1>Hi, I am Jigmat</h1>
    <p>A Full Stack Developer</p>
  </div>
  <div class="content">
    <div class="card">My Project 1</div>
    <div class="card">My Project 2</div>
  </div>
  <footer>Copyright 2026</footer>
</body>
</html>
