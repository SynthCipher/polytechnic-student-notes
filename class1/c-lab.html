Compiler Environment

Turbo C++ Setup

A classic IDE for DOS-based environments.

  • Compile: Alt + F9
  • Run: Ctrl + F9
  • Output: Alt + F5

GCC Compiler

Modern industry-standard compiler for Linux/Windows.

  • Compile: gcc file.c -o out
  • Run: ./out
  • Debug: gcc -g file.c
