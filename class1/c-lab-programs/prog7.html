Practical 7: Decision making with two paths using if-else.

Definition

The if-else statement handles two different paths. It is like a fork in the road. If the condition is True, the program follows the first path (if block). If the condition is False, it MUST follow the second path (else block). This ensures that your program always has a plan for both "Yes" and "No" answers.

Program Example

#include <stdio.h>

int main() {
    int n;
    printf("Enter a number: ");
    scanf("%d", &n);
    if(n % 2 == 0) {
        printf("The number is Even.\n");
    } else {
        printf("The number is Odd.\n");
    }
    return 0;
}

Expected Output

Enter a number: 7
The number is Odd.
