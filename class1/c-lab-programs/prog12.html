Practical 12: Storing data in matrix format using 2D arrays.

Note & Definition

A 2D array is an array of arrays, representing data in a table format with rows and columns. It is commonly used for mathematical matrices and grid-based data structures. Accessing an element requires two indices: [row_index][column_index].

Program Example

#include <stdio.h>

int main() {
    int table[2][2] = {{1, 2}, {3, 4}};
    printf("Element at [0][0]: %d\n", table[0][0]);
    printf("Element at [1][1]: %d\n", table[1][1]);
    return 0;
}

Expected Output

Element at [0][0]: 1
Element at [1][1]: 4

Homework / Practice Question

Write a program to store a 2x2 matrix and print all its elements in a proper row-column format using nested loops.

How to Compile & Run

Modern GCC
