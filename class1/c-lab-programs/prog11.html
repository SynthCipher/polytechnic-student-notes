Practical 11: Storing multiple items in a single variable using arrays.

Definition

An array is a collection of elements of the same data type stored in contiguous memory locations. A 1D array is accessed using a single index, starting from 0. They are essential for handling lists of data (like marks of 50 students) efficiently without declaring 50 different variables.

Real-Life Analogy

An array is exactly like a row of lockers in your college. All lockers are the same size, but each locker has a unique number (Index) starting from 0. To get your books, you just need the locker number. Similarly, in C, you just need the Index number to get the data from the array.

Program Example

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    printf("Value at index 0: %d\n", arr[0]);
    printf("Value at index 1: %d\n", arr[1]);
    printf("Value at index 2: %d\n", arr[2]);
    return 0;
}

Expected Output

Value at index 0: 10
Value at index 1: 20
Value at index 2: 30

Homework / Practice Question

Create an array of 5 integers, take values from the user using a loop, and print the total sum of all elements.
