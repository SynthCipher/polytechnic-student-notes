Practical 1: Simple program to start.

Definition

Every C program must follow a specific structure to be understood by the computer. The #include <stdio.h> part is called a Preprocessor Directive. It tells the compiler to load the "Standard Input Output" library. The int main() is the entry point, and return 0 indicates success.

Real-Life Analogy

Think of a C program like a formal letter. The Header is the sender's address, the Main function is the body where you write the message, and Return 0 is the formal closing like "Yours Sincerely".

Program Example

#include <stdio.h>

int main() {
    printf("Welcome to C Programming Lab!\n");
    return 0;
}
