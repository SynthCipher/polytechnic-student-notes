Practical 6: Simple decision making using the 'if' statement.

Definition

The if statement is the most basic way to make a program "smart." It works like a gatekeeper. It checks a condition (like is age >= 18?), and only if that condition is True, it allows the code inside the curly braces { } to run. If the condition is False, the code is simply ignored.

Real-Life Analogy

An if statement is like checking the sky before leaving home. IF it is raining, you take an umbrella. If it is NOT raining, you don't do anything special. The decision depends entirely on the condition of the sky.

Program Example

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    if(age >= 18) {
        printf("You are eligible to vote!\n");
    }
    return 0;
}

Expected Output

Enter your age: 20
You are eligible to vote!

Homework / Practice Question

Write a program that takes a temperature as input and prints "It's hot!" if the temperature is above 35 degrees.
