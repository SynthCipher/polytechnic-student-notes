Practical 9: Implementing post-test loops using 'do-while'.

Note & Definition

The 'do-while' loop is a post-test loop, meaning the condition is checked AFTER the loop body executes. This guarantees that the loop runs at least once, regardless of whether the condition is true or false. It is useful when the initial execution is required before testing.

Program Example

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("%d ", i);
        i++;
    } while(i <= 3);
    return 0;
}

Expected Output

1 2 3

Homework / Practice Question

Write a program using do-while to print numbers from 10 down to 1 (reverse order).
