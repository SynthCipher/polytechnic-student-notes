Storing
Numbers

Practical 2: Learning about data types and variable initialization.

Definition

Variables are like small storage boxes in memory. In C, you must declare the type of box (int, float, char). The '=' sign is the assignment operator, which puts a value inside the box.

Real-Life Analogy

Variables are like labeled jars in a kitchen. You have a jar for salt (int), a jar for oil (float), and a jar for a spoon (char). You label them so you don't mix up the items inside.

Program Example

#include <stdio.h>

int main() {
    int age = 20;
    float marks = 85.5;
    printf("Age: %d, Marks: %.1f\n", age, marks);
    return 0;
}

Expected Output

Age: 20, Marks: 85.5

Homework / Practice Question

Create variables for your 'Year' and 'Percentage' and print them with labels.
