Practical 14: Direct memory access using pointers.

Definition

Pointers are special variables that do not store regular values like 10 or 20. Instead, they store the Memory Address of another variable. They are like a GPS for your computer's memory. Pointers are incredibly powerful because they allow you to directly access and modify data in the computer's RAM, which makes programs run faster.

Real-Life Analogy

A pointer is exactly like a Treasure Map. The map itself is not the treasure, but it tells you exactly WHERE the treasure is buried in the ground (Memory). If you follow the address on the map (dereferencing), you will find the actual treasure (the value).

Program Example

#include <stdio.h>

int main() {
    int n = 100;
    int *ptr = &n;
    printf("Value of n: %d\n", n);
    printf("Address of n: %p\n", (void*)&n);
    printf("Value using pointer: %d\n", *ptr);
    return 0;
}

Expected Output

Value of n: 100
Address of n: 0x7ffd...
Value using pointer: 100

Homework / Practice Question

Create two variables, use pointers to add them together, and print the result using only the pointer variables.
