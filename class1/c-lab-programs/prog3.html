Practical 3: Using arithmetic and relational operators.

Definition

Arithmetic operators (+, -, *, /, %) perform basic math. The % (modulus) operator is unique because it gives the remainder of a division. Relational operators (>, <, ==) are used to compare values. In C, if a comparison is True, it returns 1; if it is False, it returns 0.

Program Example

#include <stdio.h>

int main() {
    int a = 15, b = 4;
    printf("Sum: %d\n", a + b);
    printf("Is a greater than b? %d\n", a > b);
    return 0;
}

Expected Output

Sum: 19
Is a greater than b? 1
