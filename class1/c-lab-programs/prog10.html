Practical 10: Using 'for' loops for controlled repetition.

Definition

The for loop is the most compact and popular loop in C. It puts three important things in one line: Initialization (starting value), Condition (how long to run), and Increment (the step). It is perfect for cases where you know exactly how many times you want to repeat a task.

Real-Life Analogy

A for loop is like an alarm clock that is set to repeat for 5 days. You tell it when to start (Day 1), how long to repeat (Until Day 5), and to move forward one day at a time. It will keep ringing until it reaches the 5th day and then stop automatically.

Program Example

#include <stdio.h>

int main() {
    int i;
    for(i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}

Expected Output

1 2 3 4 5

Homework / Practice Question

Use a for-loop to print the multiplication table of 2 (2, 4, 6... 20).
