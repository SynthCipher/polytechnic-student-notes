Practical 13: Using 'struct' to group variables of different types.

Note & Definition

Structures (struct) are user-defined data types that allow grouping variables of different data types under a single name. This is useful for representing complex entities like 'Student' (with ID and Marks) or 'Book'. The . (dot) operator is used to access structure members.

Program Example

#include <stdio.h>

struct Student {
    int id;
    float marks;
};

int main() {
    struct Student s1 = {101, 85.5};
    printf("ID: %d, Marks: %.1f\n", s1.id, s1.marks);
    return 0;
}

Expected Output

ID: 101, Marks: 85.5
