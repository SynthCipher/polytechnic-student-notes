Practical 4: Operator precedence and expression evaluation.

Note & Definition

Expression evaluation in C follows the rules of operator precedence and associativity (similar to BODMAS). Parentheses () have the highest priority, followed by multiplication/division, and then addition/subtraction. Using brackets ensures the math is solved in your desired order.

Program Example

#include <stdio.h>

int main() {
    int result = (10 + 5) * 2 / 5;
    printf("Result: %d\n", result);
    return 0;
}

Expected Output

Result: 6

Homework / Practice Question

Evaluate the expression: 5 + 10 * 2 / (4 - 2). Write a C program to verify the mathematical result.
